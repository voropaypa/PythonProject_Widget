# PythonProject_Widget
Проект, реализующий виджет, который показывает несколько последних успешных банковских операций клиента.

### Использование
Мой проект можно будет найти в моем личном аккаунте github в публичном доступе.

### Реализованные на данный момент функции:

- Получение информации о счете и его номере или о названии карты и ее номере. 
Отображение на выходе функции счета/названия карты и маску номера.

Пример входных данных для проверки функции, при получении данных карты:

`Visa Platinum 7000792289606361`

Пример данных на выходе из работы функции:

`Visa Platinum 7000 79** **** 6361`

- Сортировка банковских операций по статусу или дате их выполнения.

Пример входных данных для проверки функции:

`[{'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'}, {'id': 939719570, 'state': 'EXECUTED', 'date': '2018-06-30T02:08:58.425572'}, {'id': 594226727, 'state': 'CANCELED', 'date': '2018-09-12T21:27:25.241689'}, {'id': 615064591, 'state': 'CANCELED', 'date': '2018-10-14T08:21:33.419441'}]`

Пример данных на выходе из работы функции, при базовой сортировке по дате:

`[{'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'}, {'id': 615064591, 'state': 'CANCELED', 'date': '2018-10-14T08:21:33.419441'}, {'id': 594226727, 'state': 'CANCELED', 'date': '2018-09-12T21:27:25.241689'}, {'id': 939719570, 'state': 'EXECUTED', 'date': '2018-06-30T02:08:58.425572'}]`


### Тестирование реализованных на данный момент функций:

С помощью фреймворка pytest было проведено тестирование всех реализованных на данный момент функций.

Для каждой функции была проведена проверка работы при

- Пустом вводе данных там, где ожидается аргумент

- Вводе данных неправильного или неожидаемого формата

- При вводе примеров ожидаемых данных 

- При вводе неправильных данных, где ожидается ошибка
